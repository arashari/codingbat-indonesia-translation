# intMax

Given three int values, a b c, return the largest.

> diberikan 3 nilai `int`, a b c, kembalikan nilai terbesar di antara mereka

```
intMax(1, 2, 3) → 3
intMax(1, 3, 2) → 3
intMax(3, 2, 1) → 3
```
