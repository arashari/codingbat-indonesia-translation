# frontBack

Given a string, return a new string where the first and last chars have been exchanged.

> diberikan sebuah string, kembalikan nilai string yang baru di mana karakter pertama dan terakhirnya sudah ditukar

```
frontBack("code") → "eodc"
frontBack("a") → "a"
frontBack("ab") → "ba"
```
