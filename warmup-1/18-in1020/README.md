# in1020

Given 2 int values, return true if either of them is in the range 10..20 inclusive.

> diberikan 2 nilai `int`, kembalikan nilai `true` jika ada diantara keduanya yang berada pada rentang 10..20 inklusif

```
in1020(12, 99) → true
in1020(21, 12) → true
in1020(8, 99) → false
```
