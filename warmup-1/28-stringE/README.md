# stringE

Return true if the given string contains between 1 and 3 'e' chars.

> kembalikan nilai `true` jika diberikan string yang mengandung antara 1 sampai 3 karakter 'e'

```
stringE("Hello") → true
stringE("Heelle") → true
stringE("Heelele") → false
```
